# noetl commit subject format for automation
- Timestamp: 2026-03-17T05:08:36Z
- Author: Kadyapam
- Tags: noetl,release,automation,commit-convention,policy

## Summary
Added policy note: NoETL repository release automation requires plain conventional commit prefix without scope braces. Use 'fix: ...' (and similar types) instead of scoped forms like 'fix(runtime): ...', otherwise automation is not triggered.
