## Phase B ‚Äî implementation

- **`noetl/core/runtime/keda.py`** (new):
  - Module-level defaults exported as constants:
    `DEFAULT_LAG_THRESHOLD = 10`,
    `DEFAULT_ACTIVATION_LAG_THRESHOLD = 1`,
    `DEFAULT_POLLING_INTERVAL_SECONDS = 10`,
    `DEFAULT_COOLDOWN_SECONDS = 30`,
    `DEFAULT_MIN_REPLICAS = 1`, `DEFAULT_MAX_REPLICAS = 20`,
    `DEFAULT_NATS_STREAM = "NOETL_COMMANDS"`,
    `DEFAULT_NATS_MONITORING_ENDPOINT = "nats.nats.svc.cluster.local:8222"`,
    `DEFAULT_NATS_ACCOUNT = "$G"`.
  - `worker_pool_segment(urn)` ‚Äî extracts the last NATS-safe
    segment by routing through `to_nats_subject()` then
    splitting on `.`. Tested with both a clean URN and a
    `cpu pool`/`r&d` URN that exercises the lossy collapse.
  - `_derive_consumer_name(urn)` (private) ‚Äî URN subject body
    with `.` ‚Üí `_`. NATS consumer names disallow `.`, so the
    collapse is mandatory.
  - `ScaledObjectSpec` frozen dataclass with the 14 fields
    enumerated in the prompt. Two are `Optional[str]` because
    they default to URN-derived values when `None`.
  - `build_worker_scaledobject(spec)`:
    - Parses URN; raises `ValueError` if `kind != "tenant"`.
    - Validates `min_replicas >= 0`, `max_replicas >= min_replicas`,
      `lag_threshold > 0`, `activation_lag_threshold >= 0`,
      `polling_interval_seconds > 0`, `cooldown_seconds > 0`.
      Each violation raises `ValueError` with the offending
      field name in the message (regex-matchable in tests).
    - Derives `scaler_name` ‚Üí `<deployment>-scaler-<pool-segment>`
      and `nats_consumer` ‚Üí URN-subject-body with `.` ‚Üí `_`
      when those fields are `None`.
    - Returns a dict with `apiVersion: keda.sh/v1alpha1`,
      `kind: ScaledObject`, full metadata + spec block, single
      `nats-jetstream` trigger. Every trigger-metadata value
      is stringified (KEDA requirement, including for numeric
      lag thresholds).
  - `dump_scaledobject_yaml(d)` ‚Äî
    `yaml.safe_dump(d, sort_keys=False, default_flow_style=False)`.
    Preserves the `apiVersion ‚Üí kind ‚Üí metadata ‚Üí spec` order
    set by the generator.
  - `__all__` exports every public symbol (defaults + dataclass +
    helpers).

- **`ci/manifests/keda/scaledobject-worker-cpu-01.yaml`**
  generated by:
  ```python
  spec = ScaledObjectSpec(
  ```

  surface the tenant in the scaler name only if the URN ends
