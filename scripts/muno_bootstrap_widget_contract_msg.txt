Widget catalogue (23 templates + 1 envelope)
--------------------------------------------
Plain text: bot_text, user_text, typing_indicator, clarify_question.
Interactive inputs: date_range_picker, party_picker,
                    place_autocomplete_input, action_chooser.
Lists: flight_list, hotel_list, place_list.
Cards: flight_card, hotel_card, hotel_compare, place_card.
Layout: map_view, filter_panel, property_block.
Lifecycle: loading_card, error_card, notification.
Closure: itinerary_summary, order_confirmation.

Each template's schema explicitly enumerates:
  - Required + optional payload fields.
  - Available `variant` ids (e.g., compact | full | in_popover).
  - CTAs that the renderer should wire up (e.g., watch_in_detail).
  - ai_adjustments knobs (emphasis slots, conditional banners,
    annotation array).

Envelope (`_envelope.schema.json`):
```
{ widget_type, variant, payload, ai_adjustments?, schema_version }
```
